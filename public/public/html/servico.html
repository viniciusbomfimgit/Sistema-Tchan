Consultar um cliente

1. Acesse a lista de clientes através do botão Clientes na página inicial;

2. Busque pelo cliente desejado na caixa de busca;

3. Selecione - entre os resultados exibidos - o que corresponde corretamente ao desejado.

Consultar um agendamento

FORMA 1

1. Acesse a lista de agendamentos através do botão Agendamentos na página inicial;

2. Busque pelo agendamento desejado.

FORMA 2

1. Acesse o perfil do cliente ou colaborador envolvido no agendamento;

2. Selecione o agendamento desejado na tabela existente em seu perfil.

Consultar um serviço

1. Acesse a lista de serviços através do botão Serviços na página inicial;

2. Busque pelo serviço desejado.

Consultar um colaborador

1. Acesse a lista de colaboradores através do botão Colaboradores na página inicial;

2. Busque pelo colaborador desejado.


Retornar à tela de seleção

Cadastrar um cliente

1. Acesse a lista de clientes através do botão Clientes na página inicial;

2. Clique no botão Cadastrar cliente (No canto inferior direito da tela, representado pelo ícone "+");

3. Preencha o formulário com os dados do cliente;

4. Clique em cadastrar e, se tudo tiver sido preenchido corretamente, você receberá uma mensagem informando que ele foi cadastrado com sucesso.

Cadastrar um agendamento

1. Acesse a lista de clientes através do botão Clientes na página inicial;

2. Acesse o perfil do cliente que deseja agendar (se ele não possuir cadastro, deve ser efetuado seu cadastro antes de prosseguir);

3. Clique no botão Cadastrar agendamento (No canto inferior direito da tela, representado pelo ícone "+");

4. Seleciona a data em que ocorrerá o serviço;

5. Preencha o formulário com os serviços a serem realizados, o horário a serem realizados e, se necessário, adicione uma observação;

6. Clique no botão Confirmar e, se tudo tiver sido preenchido corretamente, você receberá uma mensagem informando que ele foi cadastrado com sucesso.

Cadastrar um serviço

1. Acesse a lista de serviços através do botão Serviços na página inicial;

2. Clique no botão Cadastrar serviço (No canto inferior direito da tela, representado pelo ícone "+");

3. Preencha o formulário com os dados do serviço;

4. Clique em cadastrar e, se tudo tiver sido preenchido corretamente, você receberá uma mensagem informando que ele foi cadastrado com sucesso.

Cadastrar um colaborador (Apenas ADM)

1. Acesse as opções do sistema através do botão Opções do sistema (engrenagem) na página inicial;

2. Clique no botão '+ Colaborador';

3. Preencha o formulário com os dados do colaborador;

4. Clique em cadastrar e, se tudo tiver sido preenchido corretamente, você receberá uma mensagem informando que ele foi cadastrado com sucesso.


Retornar à tela de seleção

Alterar um cliente

1. Utilize os passos para consultar um cliente, para chegar no perfil do cliente que deseja alterar;

2. Clique em Editar, para conseguir editar os campos do formulário;

3. Ao terminar as alterações, clique em Salvar e, se for alterado com sucesso, você receberá uma mensagem informativa;

Alterar um agendamento

1. Utilize os passos para consultar um agendamento, para chegar até o registro do agendamento que deseja alterar;

2. Efetue as alterações desejadas;

3. Ao terminar as alterações, clique em Confirmar e, se for alterado com sucesso, você receberá uma mensagem informativa;

Alterar um serviço

1. Utilize os passos para consultar um serviço, para chegar até o registro do serviço que deseja alterar;

2. Clique em Editar, para conseguir editar os campos do formulário;

3. Ao terminar as alterações, clique em Salvar e, se for alterado com sucesso, você receberá uma mensagem informativa;

Alterar um colaborador (Apenas ADM)

1. Utilize os passos para consultar um colaborador, para chegar no perfil do colaborador que deseja alterar;

2. Clique em Editar, para conseguir editar os campos do formulário;

3. Ao terminar as alterações, clique em Salvar e, se for alterado com sucesso, você receberá uma mensagem informativa;

Alterar PIN de um colaborador

1. Acesse as opções do sistema através do botão Opções do sistema (engrenagem) na página inicial;

2. Clique no botão Alterar PIN;

3. Selecione o colaborador que deseja alterar o PIN;

4. Digite o novo PIN do colaborador e o confirme;

5. Digite a senha de administrador;

6. Clique em Alterar, você receberá uma mensagem informativa sobre a conclusão da alteração.


Retornar à tela de seleção

3. Clique em Cancelar e, se for cancelado com sucesso, você receberá uma mensagem informativa;
